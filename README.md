# Add Recent Tags

An Anki add-on that lets you quickly add tags from your most recently created notes to any card, with the ability to modify tags on-the-fly before adding them.

## Features

- **Quick Access to Recent Tags**: Displays the 5 most recently used tags from your newest notes
- **Modify Before Adding**: Edit any tag before adding it to your notes

## Usage

### Accessing the Tool

Right-click to access "Add Recent Tags" from:

1. **Browser**: Right-click on selected card(s) in the card list
2. **Editor**: Right-click in the editor sidebar
3. **Reviewer**: Right-click on the card display area OR click the "More" button (⋮)

### Adding Tags

1. Click "Add Recent Tags" from the context menu
2. A dialog appears showing your 5 most recent tags
3. **Click a tag button** to toggle selection (turns green when selected)
4. **Click "Modify"** next to any tag to edit it:
   - The tag button becomes an editable text field
   - Modify the tag text as desired
   - Press Enter, click outside, or click "OK" to finish editing
   - The modified tag is automatically selected (green)
5. Click the main **OK button** to add all selected tags to your note(s)

## Configuration

Access configuration via: **Tools → Add-ons → Add Recent Tags → Config**

### Settings

- **Number of recent tags to show**: (1-20, default: 5)
  - How many tag buttons appear in the dialog
  
- **Number of recent notes to search**: (10-500, default: 50)
  - How many of your most recent notes to search through for tags
  - Higher numbers provide more tag variety but may be slower

- Click **Restore Defaults** to reset all settings to their original values.
